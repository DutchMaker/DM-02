# KiCad projects

KiCad EDA is used for all circuit diagrams and PCB design.
Get it for free at: [http://kicad-pcb.org](http://kicad-pcb.org)

### KiCad setup:

- Add `./Libraries/*.lib` as global Symbol library to KiCad (Eeschema)
- Add the `./Libraries` folder as global Footprint library to KiCad (Pcbnew)

### Notes:

- The _FreeRouting_ application (`./Libraries/freeRouting.jar`) is used for initial PCB routing
